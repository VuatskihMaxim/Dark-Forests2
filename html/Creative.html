Описание

Версия игры: 1.19.2

Версия мод пака: 1.3

Creative - это уникальный сервер для строительства, где ваши способности расширены с помощью специальных модов и плагинов.
Сервер позволяет вам почувствовать себя настоящим творцом, что кропотливо, кубик к кубику, выстраивает невероятный шедевр. На сервере помимо стандартных возможностей креатива, есть специальные инструменты для изменения структуры блоков; новые блоки, что смогут улучшить вашу постройку и сделать её непохожей на какую-либо другую; новая система механизмов.

Содержимое мод пака

Моды: Chisels & Bits - For Forge, Create, Flywheel, Soory Chimneys, MrCrayfish's Furniture Mod, Macaw's Windows, Macaw's Doors, Macaw's Trapdoors, MC Paint, Extended Creative Inventory, Prefab, Effective (Forge), Fantasy's Furniture, Biomes O' Plenty, Passable Foliage (Forge), Blue's Dynamic Lights.

Библиотеки: Fabric API, Collective, ApexCore, TerraBlender (Forge), Kiwi (Forge), Cloth Config API (Fabric/Forge).

Оптимизаторы: OptiFine, Ai Improverments, Better Fps - Render Distance[Forge].

Текстуры: Flesh Animations, Snod Animated Torches, Spawn Egg 3D, Uncontrolled Glass.
